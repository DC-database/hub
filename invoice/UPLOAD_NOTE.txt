IBA 11.0.9 FINAL UPLOAD NOTE

Purpose:
- Final privacy/safety correction after 11.0.8.
- Normal live users/admins must not see Test Mode or Firebase Active prompts.
- Test Mode remains available only for local/localhost Super Admin UI testing.

What is included:
1. Chrome-only browser guard remains active.
   - Edge, Firefox, Safari, Opera, etc. are blocked even for local/test mode.
   - Unsupported browser screen shows Copy Link for Chrome.

2. One active Chrome tab guard remains active.
   - Only one active IBA tab runs.
   - Duplicate tabs are blocked/paused and do not run Firebase-heavy sync.

3. Local Test Mode / Firebase Blocker remains active for local testing only.
   - file:/// local folder = Test Mode ON, Firebase OFF.
   - localhost / 127.0.0.1 = Test Mode ON, Firebase OFF.
   - Enter Test Preview allows UI checking without real login/database.
   - Real save/search/update/delete remains blocked in Test Mode.

4. Live system privacy correction.
   - Normal live GitHub link is always LIVE mode.
   - No LIVE MODE / Firebase Active badge is shown.
   - No Test Mode / Test Preview option is shown on live.
   - ?testmode=1 on live is ignored so users/admins cannot see or use Test Mode from live.

5. Desktop clock remains included.
   - Desktop-only numbered analog clock replaces the old name/email/profile card area.
   - No square background behind clock.
   - Mobile layout remains unchanged.

Files in this patch:
- index.html
- app.js
- version.json
- js/app-test-mode.js
- js/app-tab-guard.js
- js/app-time-clock.js
- js/app-data-cache.js

Upload instruction:
- Upload/replace the included files in your live system.
- Keep 11.0.6 as rollback if needed, but 11.0.9 is intended as the clean final version after the 11.0.8 corrections.
