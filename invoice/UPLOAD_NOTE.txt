IBA SYSTEM PATCH 11.0.5
WorkDesk + Invoice Management Firebase Download Reduction Patch

UPLOAD THESE FILES:
1. index.html
2. app.js
3. js/app-workdesk-dashboard.js
4. js/app-active-tasks.js
5. js/app-data-cache.js
6. js/app-payments.js
7. version.json
8. UPLOAD_NOTE.txt

WHAT CHANGED:
- WorkDesk Dashboard All Active no longer downloads the full invoice_entries tree.
- Dashboard All Active now reads the compact invoice_tasks_by_user/All index and validates each visible active invoice using exact invoice_entries/{PO}/{invoiceKey} reads.
- WorkDesk Active Task no longer scans full invoice_entries for personal invoice tasks.
- Active Task now reads only the logged-in user/delegated user task inbox and validates exact invoice records.
- Payment modal PO search now fetches only invoice_entries/{PO}, not the whole invoice_entries database.
- Added a safe light-data option in app-data-cache.js so modules can load PO/site/vendor data without forcing the full invoice tree.
- Added console warning when a true full invoice_entries read is requested, so heavy pages can be identified.

NOT CHANGED:
- Invoice saving logic
- Firebase write paths
- Status workflow
- Attention routing
- Job Records rules
- Inventory logic
- Financial Report calculations
- Invoice Records broad search accuracy

IMPORTANT:
- Full invoice_entries reads are still allowed for broad Invoice Records searches, Summary Note, Financial Reports, exports, and other reports where full historical data is required for accuracy.
- Dashboard and Active Task should now be much lighter for users who leave WorkDesk open all day.
